## Overview

python-highcharts-df is a wrapper around [python-highcharts](https://github.com/kyper-data/python-highcharts) for rendering customizable pretty javascript plots directly from dataframes.  Styling is done with matplotlib like key word arguments, while plots are fully customizable using the underlying [python-highcharts](https://github.com/kyper-data/python-highcharts) dictionary option tools.


## Installation:

python-highcharts-df currently supports Python 2.7. To install and contribute:

```
$ git clone https://github.com/aloosley/python-highcharts-df.git
$ pip install -e .
```
